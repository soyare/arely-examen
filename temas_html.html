1. Encabezados

Los encabezados en HTML se utilizan para resaltar títulos de secciones o subsecciones en una página. Se utilizan etiquetas como <h1>, <h2>, etc.

2. Párrafos

Un párrafo en HTML se define con la etiqueta <p>. Es una de las estructuras más comunes para separar el contenido de texto.

3. Formateo de texto

En HTML, puedes usar etiquetas como <b>, <i>, y <u> para aplicar negrita, cursiva y subrayado al texto, respectivamente.

4. Fuentes

HTML permite cambiar el tipo de fuente con la etiqueta <font> (aunque actualmente se recomienda el uso de CSS).

5. Los tres tipos principales de listas

  • Numeradas: Usando la etiqueta <ol>.
  • Sin numerar: Usando la etiqueta <ul>.
  • Glosario: Usando la etiqueta <dl> para definiciones.

6. Manejo de gráficas

El manejo de gráficas en HTML se puede hacer mediante el uso de imágenes (<img>) y atributos como align para alineación, border para márgenes, y background para establecer gráficos de fondo.

7. Tipos de enlaces

En HTML, existen tres tipos principales de enlaces:

  • A otro sitio web: Usando <a href="http://..."></a>.
  • A otra página dentro del mismo sitio: Usando un enlace relativo.
  • A otro punto dentro de la misma página: Usando <a href="#id_del_punto"></a>.

8. Creación de un formulario

Los formularios en HTML se crean con la etiqueta <form>. Los elementos que puedes usar incluyen:

  • Cajas de texto: <input type="text">
  • Cajas de verificación: <input type="checkbox">
  • Área de texto: <textarea>
